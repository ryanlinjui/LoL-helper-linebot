# LoL線上分析系統

想要稽查身邊打LoL雷包？那就用LoL線上分析系統吧
---------------------------------------------------
此Bot是我跟兩位學長大大所設計之臺師大通識課「運動分析與程式設計」的期末專題

- 前端UI介面 + 後端爬蟲：臺師大資工系大四 陳兆閔

- 前後端接合 + 後端分析與資料庫：臺師大資工系大一 林昕鋭

- 資料分析決策者：臺師大休旅所碩班 陳宥廷
